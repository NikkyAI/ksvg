getAttributes

open fun getAttributes(renderMode: SVG.RenderMode): Map<String, String?>

Get the attributes specific to a render mode. Allows tags to modify their attributes during rendering based on the rendering mode. Defaults to the basic Element attributes but can be overridden by Elements to return differing attribute based on mode.

Parameters

renderMode - which mode we are rendering in
